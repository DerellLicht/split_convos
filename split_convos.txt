This script was written by ClaudeAI on 07.07.26 and modified a number of times
between that date and 07.18.26
It will split an exported conversations.json file into
separate conversations in .md format.

To use it: drop your export JSON next to it, 
update the SRC path at the top if the filename's different, 
and run 'python split_convos.py conversations.json' . 
It'll split the per-conversation .md files into current directory
(or wherever you set OUT_DIR).
